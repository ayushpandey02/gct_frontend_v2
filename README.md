# GCT V2 - Goregaon Champions Trophy Website

This is the modernized frontend for the Goregaon Champions Trophy, built with React + Vite.

## Features
-   **Landing Page**: Hero section, Organisers Deck, Past Winners, Tournament Details.
-   **Registration**: Multi-step form with validation, file uploads (Cloudinary), and UPI payment info.
-   **Theme**: Premium White & Gold design.
-   **Responsive**: Mobile-first layout with bottom navigation.

## Tech Stack
-   React
-   Vite
-   Vanilla CSS (Modern)
-   React Router DOM

## Running Locally

1.  Clone the repository.
2.  Install dependencies:
    \`\`\`bash
    npm install
    \`\`\`
3.  Start the development server:
    \`\`\`bash
    npm run dev
    \`\`\`

## Backend
Ensure the backend server is running for registration functionality. Set up the `.env` file in the backend directory as per the documentation.
